# Purpose
Generate comprehensive test suites.

## Instructions
1. Analyze the code to understand functionality
2. Identify edge cases
3. Generate tests covering:
   - Happy path
   - Edge cases
   - Error conditions
   - Boundary values

4. Use appropriate testing framework (pytest, jest, etc.)

## Output
Write test files following project conventions.
